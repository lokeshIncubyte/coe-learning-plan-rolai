# User Stories

User stories for the **Task Management App**, one file per day. Each story uses **As / I want / So that** plus a **Given / When / Then** acceptance block. Backend days (3–6) are written from the API-consumer perspective; frontend days (7–9) from the end-user perspective; Day 10 mixes both.

Each day's checklist lives in [`../checklists/`](../checklists/).
